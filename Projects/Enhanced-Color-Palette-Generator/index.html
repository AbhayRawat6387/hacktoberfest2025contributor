Color Palette Generator

Generate beautiful color palettes with one click!

How to use:

  • Click "Generate New Palette" to create random colors (or press 'G')
  • Switch between HEX, RGB, and HSL formats with the format buttons
  • Click on any color to copy the current format to clipboard
  • Lock colors by clicking the lock icon to keep them in next generation
  • Save your favorite palettes for later use (or press 'S')
  • Export palettes as CSS, SCSS, JSON, text, or SVG (or press 'E')
  • Check contrast ratios for accessibility compliance
